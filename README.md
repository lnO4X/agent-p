# 搭子GG (dazi-gg)

> 🎮 "找搭子，上GG" — 游戏搭子众包平台

## What is 搭子GG?

搭子GG 是一个游戏陪玩众包平台，通过 AI Agent 对话交互，连接玩家（消费方）与搭子（服务方）。

**核心特点：**

- 🤖 AI Agent 驱动的对话式交互（发单、接单、管理）
- 👆 Tinder 式滑动匹配订单与搭子
- 🏢 公会体系管理项目与搭子
- 💰 邀请永久赏金机制
- 📊 灵活定价：平台最低价 → 公会定价 → 玩家出价

## 角色说明

| 角色 | 说明 |
|------|------|
| 管理员 (Admin) | 平台管理，维护标准项目库 |
| 公会 (Guild) | 管理项目与搭子 |
| 搭子 (Buddy) | 提供陪玩服务 |
| 玩家 (Player) | 消费方，找搭子一起玩 |

## 阶段规划

- **Phase 1** — MVP：单公会、AI对话发单、Tinder匹配、基础聊天
- **Phase 2** — 支付与结算：C2C入金、自动结算
- **Phase 3** — 多公会与运营：评价、排行、运营工具
- **Phase 4** — Agent驱动：AI管理、智能匹配、增值服务
